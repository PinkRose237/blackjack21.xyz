# Blackjack21.xyz

An interactive web application designed to help you master the game of Blackjack. Whether you're a beginner learning the basics or a player looking to refine your strategy, **Blackjack21.xyz** provides the tools you need to beat the dealer.

## 🃏 Features

-   **Interactive Strategy Trainer:** Practice real-game scenarios with a built-in strategy engine. Get instant feedback on every move—Hit, Stand, Double Down, or Split.
-   **Learn the Rules:** A comprehensive guide to Blackjack objectives, card values, and gameplay flow.
-   **Basic Strategy Insights:** Detailed explanations for every scenario to help you understand *why* a move is correct.
-   **Responsive Design:** Optimized for both desktop and mobile, so you can practice on the go.

## 🚀 Getting Started

This is a static web application. To run it locally:

1.  Clone the repository.
2.  Open `index.html` in your web browser.

## 🛠️ Built With

-   **HTML5** - Semantic structure.
-   **CSS3** - Custom styling with a classic casino aesthetic.
-   **JavaScript (ES6+)** - Interactive game logic and strategy validation.

## 📈 Strategy Engine

The practice tool follows standard "Basic Strategy" rules, including:
-   Always splitting Aces and 8s.
-   Standing on Hard 17 or higher.
-   Doubling down on 11.
-   Special rules for "Soft" hands (hands containing an Ace).
